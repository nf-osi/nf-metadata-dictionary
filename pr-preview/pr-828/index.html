About the NF Metadata Dictionary

The NF Metadata Dictionary defines the data model for the Neurofibromatosis (NF) research community, maintained by the NF Open Science Initiative (NF-OSI) Data Coordinating Center at Sage Bionetworks.

Built with LinkML, the model compiles to multiple formats including JSON-LD (for schematic validation) and JSON Schema (for Synapse platform validation).
